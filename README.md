# auth-api
This is the Authentication API's developed for Learning purpose

Tech used:

1. Node.js
2. Apollo Server
3. GraphQL
4. Prisma ORM
5. Authentication (JWT / OAuth)
6. Redis
7. PostgreSQL (local + cloud via Supabase / Railway)

Feature:
1. Sigup login
2. Refresh token rotation
3. Redish backlist
4. Google, Facebook, github Authentication
5. OTP Via Email
6. Password Reset Flow
7. Rate limiting with Redish
8. Public and Protected Route

EXPECTED OUTCOME:
Secure Auth Foundation for all kind of projects
